Компоненты шаблона.

Каждый компонент в отдельном каталоге.

Основной обязательный файл компонента - компонент.php - имя файла совпадает с каталогом.
Например _menu/_menu.php

Необязательные файлы:
	Стили - style.css (автоподкючение в секцию HEAD)
	JS — компонент.js (автоподкючение в секцию HEAD)
	Опции - options.ini и options.php

Если каталог компонента начинается с _ или -, то компонент игнорируется в админ-панели.

Ручное подключение компонента (на примере _menu):
	if ($fn = mso_fe('components/_menu/_menu.php')) require($fn);
