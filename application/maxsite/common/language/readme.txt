Языковой перевод MaxSite CMS
----------------------------
См. http://max-3000.com/page/jazykovoj-perevod-maxsite-cms

Языковые файлы
--------------
https://github.com/maxsite/Language_for_MaxSite_CMS

Установка языка
---------------
Скопируйте языковые файлы в «application/maxsite/language». 
Язык указывается в «application/maxsite/mso_config.php». 

********************************************************************************

По всем вопросам, связанным с переводом, обращаться только к автору перевода.
