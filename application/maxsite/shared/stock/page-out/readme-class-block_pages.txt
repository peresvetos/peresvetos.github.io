 * Класс для вывода блоков записе в шаблоне

Использование:

	// инициализация с начальными параметрами для получения записей
	$b = new Block_pages( array (
			... параметры ...
		));
	
	// непосредственно вывод записей
	$b->output(	array (
			... параметры ...
		));

Параметры для new Block_pages:

'limit' — количество получаеvs[ записей (всего)
'cut' — значение cut 
'pagination' — true/fasle — использовать ли пагинацию
'cat_id' — рубрики через запятую
'page_id' — номера записей через запятую 
'type' — тип записей (по-умолчанию blog)

Параметры для $b->output см. в методе output().
